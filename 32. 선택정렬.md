# 32. 선택정렬

* N개이 숫자가 입력되면 오름차순으로 정렬하여 출력하는 프로그램을 작성하세요.
정렬하는 방법은 선택정렬입니다


#### ▣ 입력설명

* 첫 번째 줄에 자연수 N(1<=N<=100)이 주어집니다.
두 번째 줄에 N개의 자연수가 공백을 사이에 두고 입력됩니다. 각 자연수는 정수형 범위 안에 있습니다.




#### ▣ 출력설명

* 오름차순으로 정렬된 수열을 출력합니다.




#### ▣ 입력예제 1

6
13 5 11 7 23 15




#### ▣ 출력예제 1

5 7 11 13 15 23


## 내가 짠 코드

```c++
#include <stdio.h>
#include <limits.h>
int main()
{
	freopen("input.txt","rt",stdin);
	int num, i, j, res;
	scanf("%d", &num);
	int temp[101];

	i = 0;
	while (i < num)
	{
		scanf("%d", &temp[i]);
		i++;
	}
	i = 0;
	while (temp[i] != '\0')
	{
		j = i + 1;
		while (temp[j] != '\0')
		{
			if (temp[i] > temp[j])
			{
				res = temp[j];
				temp[j] = temp[i];
				temp[i] = res;
			}
			j++;
		}
		i++;
	}
	i = 0;
	while (temp[i])
	{
		printf("%d ", temp[i]);
		i++;
	}
	return (0);
}

```
* 굉장히 복잡하게 짠편이다. 십의 자리인지 일의 자리인지 구별하는 것 때문에 거꾸로 갔는데 그럴 필요가 없었다.
